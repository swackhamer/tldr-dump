# git-alias

> Create shortcuts for Git commands. Part of `git-extras`. More information: <https://github.com/tj/git-extras/blob/master/Commands.md#git-alias>.

## Examples

### List all aliases

```bash
git alias
```

### Create a new alias

```bash
git alias "name" "command"
```

### Search for an existing alias

```bash
git alias ^name
```
