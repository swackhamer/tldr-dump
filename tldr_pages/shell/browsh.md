# browsh

> View webpages on the terminal using a Firefox backend. More information: <https://www.brow.sh/docs/keybindings/>.

## Examples

### Start browsh

```bash
browsh
```

### Start browsh in a specific webpage

```bash
browsh --startup-url URL
```

### Focus URL bar

```bash
<Ctrl l>
```

### Exit browsh

```bash
<Ctrl q>
```

### Display help

```bash
browsh [-h|--help]
```
