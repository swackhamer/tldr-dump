# drush

> Shell and scripting interface for Drupal. More information: <https://www.drush.org/latest/usage/>.

## Examples

### Enable module "foo"

```bash
drush en foo
```

### Uninstall module "foo"

```bash
drush pmu foo
```

### Clear all caches

```bash
drush cr
```

### Clear CSS and JavaScript caches

```bash
drush cc css-js
```
