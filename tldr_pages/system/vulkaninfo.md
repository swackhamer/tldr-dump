# vulkaninfo

> Print system Vulkan information. More information: <https://vulkan.lunarg.com/doc/view/latest/linux/vulkaninfo.html>.

## Examples

### Print full Vulkan information

```bash
vulkaninfo
```

### Print a summary

```bash
vulkaninfo --summary
```

### Make a HTML document of the full Vulkan information

```bash
vulkaninfo --html
```
