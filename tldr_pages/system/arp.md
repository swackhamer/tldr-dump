# arp

> Show and manipulate your system's ARP cache. More information: <https://manned.org/arp.8>.

## Examples

### Show the current ARP table

```bash
arp -a
```

### [d]elete a specific entry

```bash
arp -d address
```

### [s]et up a new entry in the ARP table

```bash
arp -s address mac_address
```
