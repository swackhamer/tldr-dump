# maza

> Local ad blocker. Like Pi-hole but local and using the operating system. More information: <https://github.com/tanrax/maza-ad-blocking>.

## Examples

### Update the Maza database

```bash
maza update
```

### Start Maza

```bash
sudo maza start
```

### Stop Maza

```bash
sudo maza stop
```

### Show the status of Maza

```bash
maza status
```
