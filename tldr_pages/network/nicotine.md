# nicotine

> Graphical client for the Soulseek peer-to-peer network. More information: <https://manned.org/nicotine>.

## Examples

### Start Nicotine+

```bash
nicotine
```

### Start Nicotine+ in headless ([n]) mode

```bash
nicotine [-n|--headless]
```

### Rescan shared files

```bash
nicotine [-r|--rescan]
```

### Display help

```bash
nicotine [-h|--help]
```
