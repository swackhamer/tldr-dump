# pamtopam

> Copy a PAM image. More information: <https://netpbm.sourceforge.net/doc/pamtopam.html>.

## Examples

### Copy a PAM image (i.e. a PBM, PGM, PPM or PAM image) from `stdin` to `stdout`

```bash
pamtopam < path/to/image.pam > path/to/output.pam
```

### Display version

```bash
pamtopam [-v|-version]
```
