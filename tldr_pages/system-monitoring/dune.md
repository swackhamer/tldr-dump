# dune

> A build system for OCaml programs. More information: <https://dune.build>.

## Examples

### Build all targets

```bash
dune build
```

### Clean up the workspace

```bash
dune clean
```

### Run all tests

```bash
dune runtest
```

### Start the utop REPL with compiled modules automatically loaded into it, to remove the need to load them by hand

```bash
dune utop
```
