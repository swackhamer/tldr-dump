# ppmtoppm

> Copy a PPM image. More information: <https://netpbm.sourceforge.net/doc/ppmtoppm.html>.

## Examples

### Copy a PPM image (i.e. a PBM, PGM or PPM image) from `stdin` to `stdout`

```bash
ppmtoppm < path/to/image.ppm > path/to/output.ppm
```

### Display version

```bash
ppmtoppm [-v|-version]
```
