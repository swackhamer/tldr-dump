# elinks

> A text based browser similar to `lynx`. More information: <https://github.com/rkd77/elinks>.

## Examples

### Start ELinks

```bash
elinks
```

### Quit elinks

```bash
<Ctrl c>
```

### Dump output of webpage to console, colorizing the text with ANSI control codes

```bash
elinks -dump -dump-color-mode 1 url
```
