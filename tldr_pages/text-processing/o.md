# o

> Orbiton, a simple configuration-free text editor. More information: <https://github.com/xyproto/orbiton>.

## Examples

### Open a file in editor

```bash
o path/to/file
```

### Open a file as read-only

```bash
o [-m|-monitor] path/to/file
```

### Save the file

```bash
<Ctrl s>
```

### Quit Orbiton

```bash
<Ctrl q>
```

### Display help

```bash
o [-h|--help]
```
