# anki

> Powerful, intelligent flashcard program. More information: <https://docs.ankiweb.net>.

## Examples

### Launch the GUI

```bash
anki
```

### Use a specific [p]rofile

```bash
anki -p profile_name
```

### Use a specific [l]anguage

```bash
anki -l language
```

### Use a non-default directory (`~/Anki` for default)

```bash
anki -b path/to/directory
```
