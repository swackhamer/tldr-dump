# clido

> Save-state TODO app for the terminal. More information: <https://codeberg.org/Oglo12/clido/wiki>.

## Examples

### Create a list

```bash
clido --new name
```

### Load a list

```bash
clido --load name
```

### Delete a list

```bash
clido --remove name
```

### List all lists

```bash
clido --lists
```

### Toggle autowrite

```bash
clido toggle-autowrite
```

### Open a list in a text editor

```bash
clido edit text_editor
```

### Display help

```bash
clido -h
```

### Display version

```bash
clido -v
```
