# whois

> Client for the WHOIS (RFC 3912) protocol. More information: <https://manned.org/whois>.

## Examples

### Get information about a domain name

```bash
whois example.com
```

### Get information about an IP address

```bash
whois 8.8.8.8
```

### Get abuse contact for an IP address

```bash
whois -b 8.8.8.8
```
