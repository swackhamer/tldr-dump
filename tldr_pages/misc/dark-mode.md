# dark-mode

> Control macOS dark mode. More information: <https://github.com/sindresorhus/dark-mode>.

## Examples

### Toggle dark mode (turn it on if it's currently off, off if it's currently on)

```bash
dark-mode
```

### Turn dark mode on

```bash
dark-mode on
```

### Turn dark mode off

```bash
dark-mode off
```

### Check if dark mode is on

```bash
dark-mode status
```
