# wuzz

> Interactively inspect HTTP requests and responses. More information: <https://github.com/asciimoo/wuzz>.

## Examples

### Start `wuzz`

```bash
wuzz
```

### Send an HTTP request

```bash
<Ctrl r>
```

### Switch to the next view

```bash
<Ctrl j><Tab>
```

### Switch to the previous view

```bash
<Ctrl k><Shift Tab>
```

### Display help

```bash
<F1>
```
