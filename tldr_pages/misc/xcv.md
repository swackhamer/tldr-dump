# xcv

> Cut, copy, and paste in the command-line. More information: <https://github.com/busterc/xcv>.

## Examples

### Cut a file

```bash
xcv x input_file
```

### Copy a file

```bash
xcv c input_file
```

### Paste a file

```bash
xcv v output_file
```

### List files available for pasting

```bash
xcv l
```
