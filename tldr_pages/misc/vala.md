# vala

> Vala code runner. Tutorial: <https://wiki.gnome.org/Projects/Vala/Tutorial>. More information: <https://docs.vala.dev/index.html>.

## Examples

### Run a vala file, with gtk+

```bash
vala path/to/file.vala --pkg gtk+-3.0
```

### Display help

```bash
vala --help
```

### Display version

```bash
vala --version
```
