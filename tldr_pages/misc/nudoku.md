# nudoku

> Sudoku game in terminal. More information: <https://jubalh.github.io/nudoku/>.

## Examples

### Start a sudoku game

```bash
nudoku
```

### Choose the difficulty of the game

```bash
nudoku -d easy|normal|hard
```

### Navigate the board

```bash
<h>|<j>|<k>|<l>|<ArrowKeys>
```

### Delete a number

```bash
<Backspace>|<x>
```

### Get a hint

```bash
<H>
```

### See the complete solution

```bash
<S>
```

### Create a new puzzle

```bash
<N>
```

### Quit the game

```bash
<Q>
```
