# zotero

> Manage your bibliographies. More information: <https://www.zotero.org/support>.

## Examples

### Run with the GUI

```bash
zotero
```

### Run in headless mode

```bash
zotero --headless
```

### Run with a specific profile

```bash
zotero -P profile
```

### Run the Migration Assistant

```bash
zotero --migration
```
