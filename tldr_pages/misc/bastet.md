# bastet

> Clone of the game Tetris in the terminal. More information: <https://fph.altervista.org/prog/bastet.html>.

## Examples

### Start a Tetris game

```bash
bastet
```

### Navigate the piece horizontally

```bash
<ArrowLeft>|<ArrowRight>
```

### Rotate the piece clockwise or counterclockwise

```bash
<Space>|<ArrowUp>
```

### Soft drop the piece

```bash
<ArrowDown>
```

### Hard drop the piece

```bash
<Enter>
```

### Pause the game

```bash
<p>
```

### Quit the game

```bash
<Ctrl c>
```
