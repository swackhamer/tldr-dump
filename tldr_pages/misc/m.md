# m

> Swiss Army Knife for macOS. More information: <https://github.com/rgcr/m-cli>.

## Examples

### Get the battery status

```bash
m battery status
```

### Turn off Bluetooth

```bash
m bluetooth off
```

### List available filesystems for formatting

```bash
m disk filesystems
```

### Enable Dock's auto hide feature

```bash
m dock autohide YES
```

### Disable the firewall

```bash
m firewall disable
```
