# cmd-wrapped

> View shell history statistics. More information: <https://crates.io/crates/cmd-wrapped>.

## Examples

### Fetch current statistics

```bash
cmd-wrapped
```

### Fetch statistics from a specific year

```bash
cmd-wrapped year
```

### Specify a shell

```bash
cmd-wrapped [-s|--shell] shell
```

### Display help

```bash
cmd-wrapped [-h|--help]
```

### Display version

```bash
cmd-wrapped [-V|--version]
```
