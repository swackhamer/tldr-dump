# cpan

> Query, download and build perl modules from CPAN sites. More information: <https://manned.org/cpan>.

## Examples

### Install a module (`-i` is optional)

```bash
cpan -i module_name
```

### Force install a module (`-i` is not optional)

```bash
cpan -fi module_name
```

### Upgrade all installed modules

```bash
cpan -u
```

### Recompile modules

```bash
cpan -r
```
