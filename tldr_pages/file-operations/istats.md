# istats

> Show various statistics such as the CPU temperature, fan speeds and battery status. More information: <https://github.com/Chris911/iStats>.

## Examples

### Show all the stats

```bash
istats
```

### Show all CPU stats

```bash
istats cpu
```

### Show all fan stats

```bash
istats fan
```

### Scan and print temperatures

```bash
istats scan
```
